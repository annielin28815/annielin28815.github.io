一、定義

二、使用方式

1. 在 JSX 使用 Attribute(屬性)方式傳遞下去

  注意屬性建議使用 camelCase 命名。

2. (ES6) Class Component 綁定

  在 class component，必須先在 class 的 constructor() 繼承 props 的屬性。日後在 render() 就可以直接用 this.props 呼叫傳進來的參數。

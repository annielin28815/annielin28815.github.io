2021-09-22

一、前言

  VSCode 是微軟開發的文字編輯器，同時支援 Windows、Linux 和 macOS 作業系統且開放原始碼，我自己於 2018 開始自學一點點 Html 時，是使用線上教學裡推薦的 Atom 。後來因為 VSCode 的工具跟資源較多，入職公司也是使用這套，所以較常使用。這篇主要介紹一些 VSCode 好用的快捷鍵，下一篇再介紹其他擴充之實用功能。

二、VSCode 編打程式碼 常用快捷鍵

  VSCode 這邊以 windows 作業系統為例，mac 作業系統僅要將 Ctrl 換成 Command 應該就可以了。因為在公司使用 windows 開發，回家則使用 mac 練習，我習慣開發時使用這些快捷鍵簡短用滑鼠或觸控板控制的時間，久了也順手很多。

項次 指令 用途
1 Ctrl + ` 開啟預設的 Terminal 工具
2 Ctrl + N 開啟編輯器新分頁視窗
3 Ctrl + Shift + N 開啟編輯器的全新視窗
4 Ctrl + S 另存新檔
5 Ctrl + P 快速尋找某個最近打開過的檔案
6 Ctrl + D 搜尋該檔裡某段文字
7 Ctrl + Z 恢復上一個動作
8 Alt + 方向鍵 快速移動某段已選取之文字至想要的位置
9 Ctrl + 方向鍵 快速移動到某單字的頭/尾端
10 Ctrl + Shift + 方向鍵 快速選取某段程式碼
11 Alt + Ctrl + Shift + 方向鍵 快速複製選取的某段程式碼
12 Ctrl + / 將游標停留位置或選取文字變成註解
13 tab 往後(右)縮排
14 Shift + tab 往前(左)縮排
14 F12 把游標停留在某變數上，按下後可跳到定義的行(超方便的功能!)

三、VSCode 面板使用方面快捷鍵

項次 指令 用途
1 Ctrl + + ｀ 放大(公開 code review 時很好用…)
2 Ctrl + - 縮小
3 Ctrl + N 開啟編輯器新分頁視窗
4 Ctrl + Shift + N 開啟編輯器的全新視窗
5 Ctrl + Shift + W 關閉整個編輯器
6 Ctrl + B 開啟/關閉檔案總管

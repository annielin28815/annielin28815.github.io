Day05：工程師必學的 Markdown 筆記語法

Date: 2023-11-15  23:08:08 Category: 新手全端工程師的職場成長歷程

一、前言

  大家一定都知道並且常使用 Markdown 吧？那你們常使用的工具有哪些呢？以下我就簡單介紹一些我經常使用的工具、平台與小技巧或習慣。

二、經常使用 Markdown 的推薦工具與平台

  

項次 工具或平台名稱 備註
1 Hackmd 最推團隊討論時使用。
2 Joplin 可以同步到雲端硬碟或 GitHud，適合用在私人與公用電腦上隨時切換。
3 Medium 剛踏入轉職實用的，但沒有即時預覽功能，有些語法也不支援
4 Discord 團隊之間 溝通一小段程式碼時超實用！

三、使用 Markdown 之個人小技巧或習慣

  1. 圖片插入技巧
    我個人覺得有的時候還滿適合用的，有點像在 html 語法裡為圖片加上 figacation tag 的樣子，讓看文章的人可以清楚知道圖片的敘述。

  1. 做折疊目錄時可使用
  1. 想為某段名詞或句子做些螢光記號(使用半型兩個等於將詞彙匡起來)
    我個人覺得做會議記錄時滿好用。

三、總結與自我鼓勵

  其實很多想分享的，但好像有些在這個平台沒辦法成功預覽，大家可以貼回去自己其他常用的編輯器或網站試試看實際效果唷～

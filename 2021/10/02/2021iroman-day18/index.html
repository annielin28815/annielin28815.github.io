二、HTML 裡的符號

因為我很常使用 Markdown 語法做筆記或會議記錄，甚至近期已經著迷到會用它做旅遊規劃的慾望清單 XD 所以難免在日常累積中，會有需要用到特殊符號的需求，以下分享一些我個人搜集並常常使用的網站資源。

項次 符號種類 說明
1 基本常見的字母或特殊字元 就是日常會用到的，這個網站都有唷～
2 非常齊全、並意想不到的符號 左側有功能選單，非常清楚明瞭，可以組出特別的想法們
3 常見和冷門的符號 有提供 UNICODE、HEX CODE 和 HTML CODE，是我個人最常使用的查詢網站。我覺得很多都很可愛，像是雪人 ☃
4 HTML 的數學特殊符號 像是 ∑
5 基本且常見的實用符號 像是版權的 © 或是基本的箭頭符號 →

2. 用 Calc 計算 css 變數

  • CSS 中更有一個神奇的運算方法 calc，此函式可以無視讓不同單位的數值直接做運算，如百分比與絕對單位運算（ex: 100% - 30px）。CSS Variables 也同樣能夠搭配此方法產生新的值。

  • calc 是作為函式使用，因此可將預期運算的方法置入 calc() 的括號內即可。

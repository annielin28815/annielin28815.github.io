JavaScript 的物件型別

2021-10-21

一、定義

名詞 定義
object(物件) 可以是零至多種屬性的集合，而屬性是鍵(key)和值(value)之間的關聯。
array(陣列) 一樣是零至多個元素的集合，但沒有規定其內容。是個有順序的集合。

二、object(物件)

  物件可以是零至多種屬性的集合，而屬性是鍵(key)和值(value)之間的關聯。

建立物件的方法有兩種：

1. 直接使用 {}(大括號)

  這是目前最常見的方式。而這種建立物件的方式稱為「物件實字(Object literal)」，是 JSON 格式的核心語法。

2. 透過 new 關鍵字建立一個物件實體

三、array(陣列)

  可以看作是一個特別的物件，因為一樣是零至多個元素的集合，但沒有規定它只能放什麼進去，裡面可以放原始的資料類型、另一個陣列、物件甚至是函示。

  較需注意的是，陣列是個有順序的集合，所以只能透過 [] (中括號) 加上索引來存取。

建立陣列的方法有兩種：

1. 直接使用 [] (中括號)

  。

2. 透過 new 關鍵字建立一個陣列

3. 陣列運用的其他方法： push, pop, shift, unshift 等。

四、本篇實用方法彙整

  • 在陣列和字串裡，查看內容長度的語法： length
  • 判斷某變數是否為陣列而非物件時的語法： isArray()

JavaScript 的基本型別

2021-10-17

一、定義

名詞 定義
string(字串) 會用一組 ‘’(單引號) 或 “”(雙引號) 包夾住
number(數字) 不管是整數或帶有小數點的浮點數字都是 number
boolean(布林值) 只有 true 和 false 兩種。
null(空值) 指某變數可能曾經有或曾經沒有值，但現在「沒有」值。
undefined(未給值) 尚未給值，所以不知道是什麼。
template literal(樣板字面值) 是由一組 `(反引號)包覆，且由一般字串與 $(錢字號)、 {}(大括號)所組成。

二、string(字串)

  string(字串)會用一組 ‘’(單引號) 或 “”(雙引號) 包夾住，兩種不能混用或包含著一起使用，在適當時候可使用 \ (跳脫字元)與 +(加號)運用。

三、number(數字)

  不管是整數或帶有小數點的浮點數字都是 number(數字) ，比較特別的還有 Infinity(正無限大)、-Infinity(負無限大)和 NaN(不是數值，Not A Number)。

四、boolean(布林值)

  只有 true 和 false 兩種。通常使用在判斷式，作為控制程式流程的用途。

五、null

  係指某變數可能曾經有或曾經沒有值，但現在「沒有」值。

六、undefined

  係指尚未給值，所以不知道是什麼。

七、template literal(樣板字面值)

  ES6 開始才有的，為 JavaScript 提升字串的靈活使用度。是由一組 `(反引號)包覆，且由一般字串與 $(錢字號)、 {}(大括號)所組成。

八、本篇實用方法彙整

  • 判斷型別的語法： typeof
  • 強制轉型的語法： Number()
  • 檢查某個變數是否為 NaN 的語法： isNaN()
  • \ (反斜線) 後面不能有任何東西，包含空白字元。
